Background Image Effect

Description of the zoom Effect



I chose the zoom trick for background images. First we specify the dimensions for the parent element. Then the child can fill the parent using width: 100% and height: 100%;, as well as set the background image, ensuring it scales to cover the area. To finish up the basic effect, we can add some transitions to our child element's normal state. We then add hover effects to our parent element which will affect our child element.
